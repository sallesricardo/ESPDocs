## Introdução

O ESP-IDF é um conjunto de ferramentas e bibliotecas para desenvolvimento de aplicações para o ESP32 e ESP32-S2.

## Instalação

Para instalar o ESP-IDF, siga as instruções no site oficial do ESP-IDF:

https://docs.espressif.com/projects/esp-idf/en/latest/esp32/get-started/index.html
